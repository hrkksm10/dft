# DFT

    simple DFT
    (C extension module)

## Installation

Add this line to your application's Gemfile:

    gem 'dft', :path => 'vendor/gems/dft'

And then execute:

    $ bundle

## Usage

    DFT.forward(values, window)
    * values: target data array
    * windows: windowing data array(option)
